This app is configured to automatically deploy to GitHub Pages using GitHub Actions.

### Setup Instructions

1. Go to your GitHub repository settings: `https://github.com/vic-odes/val/settings/pages`
2. Under "Build and deployment", set:
   - **Source**: GitHub Actions
3. Push changes to the `main` branch to trigger deployment
4. Your app will be available at: `https://vic-odes.github.io/val/`

### Manual Deployment

You can also trigger a manual deployment by:
1. Going to the "Actions" tab in your repository
2. Selecting the "Deploy to GitHub Pages" workflow
3. Clicking "Run workflow"
